# 智能呼叫中心 - 版本更新日志

## v1.1.6 (2026-04-14)

### 新功能
- **客户列表Tab切换**: 任务详情页客户列表支持Tab切换（全部/待拨打/已拨打）
- **自动拨号暂停恢复**: 暂停拨号后，下次可从上次位置继续拨打
- **拨号完成自动标记**: 电话挂断后自动标记客户为已拨打状态

### 优化
- **无权限拨号处理**: 无READ_PHONE_STATE权限时，仍可正常标记拨号完成
- **客户列表实时刷新**: 拨号完成后客户列表自动刷新，显示最新状态
- **Tab状态保持**: 拨号完成后保持当前Tab位置，不自动切换
- **状态管理优化**: 将Tab选中状态提升到父组件，确保刷新后状态不丢失

### 修复
- 修复自动拨号标记完成后不刷新列表的问题
- 修复无通话状态权限时自动拨号中断的问题
- 修复Tab自动切换的问题
- 修复Tab状态在数据刷新后丢失的问题

## v1.1.5 (2026-04-03)

### 新功能
- **客户列表整合**: 客户Tab现在显示所有任务中的客户 + 分配给自己的客户
- **自动拨号配置**: 新增自动拨号配置对话框，支持设置：
  - 拨号间隔（3-30秒）
  - 通话超时时间（10-120秒）
  - 每个客户连续拨打次数（1-5次）
- **客户去重**: 同一客户存在于多个任务中时自动去重显示
- **任务客户编辑**: 支持在任务执行页面编辑客户信息和删除客户
- **通话中客户信息**: 自动拨号时，在屏幕顶部固定显示当前和下一个客户信息
- **自动拨号状态标记**: 自动拨号完成后自动标记客户为已拨打状态
- **客户列表分组显示**: 任务详情页客户列表按"待拨打"和"已拨打"分组显示
- **通知栏客户信息**: 自动拨号时，通知栏显示当前拨打客户信息

### 优化
- **自动拨号进度保存**: 应用退出后自动保存拨号进度，重启可恢复
- **客户列表搜索**: 支持按姓名、电话、公司搜索客户
- **客户状态筛选**: 支持按待拨打/已完成状态筛选
- **任务详情页UI**: 删除中间的进度条，简化界面
- **删除客户性能**: 优化删除客户时的加载速度，不再重新加载整个任务

### 修复
- 修复客户Tab无数据的问题
- 修复自动拨号权限检查问题
